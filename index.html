Conceito Web

A palavra inglesa "web" significa "teia" ou "rede". Com o surgimento da internet, o significado do termo se transformou. Atualmente, o termo "web" refere-se a uma rede que conecta computadores em todo o mundo.

O que é o WWW?

É a abreviação da frase "web mundial", que significa "rede com amplitude mundial". Esse é o método pelo qual as informações em hipertexto circulam. É composto por milhões de computadores e sites conectados.

O que é W3C?

O W3C é o que? O World Wide Web Consortium é uma organização global composta por organizações independentes, órgãos governamentais e empresas privadas. Os membros da World Wide Web Community (W3C) trabalham para criar protocolos e modelos de padronização para a rede mundial de computadores.

O que é um servidor?

Um servidor pode ser definido como qualquer dispositivo tecnológico capaz de executar programas de forma centralizada, armazenar e compartilhar arquivos, controlar filas de impressão e fornecer vários serviços para vários computadores conectados a uma rede local ou pública.
